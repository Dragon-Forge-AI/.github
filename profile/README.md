## 🧠 The Awen Philosophy

> *"Free lines. No hard right-angles. No rigid chines. Forms that flow like water, move like predators, resonate like sound."*

The Awen is the Welsh concept of flowing inspiration — the three rays of light that represent the union of past, present, and future. It is our design law. Every hull curve, every UI, every line of code must embody it.

It is not an aesthetic. It is a constraint.
